> *We spoon-feed your business directly to ChatGPT.*

## About the Project

ChatGPT is answering millions of questions every day. When someone asks it *"who's the best accountant in Cape Town?"* or *"which marketing agency should I use?"*, it doesn't Google it — it pulls from websites it can actually read and understand.

If your site isn't structured for AI, it gets skipped. Your competitor — who did set it up correctly — gets recommended instead. Every single time.

**Thunderspoon** is an AI visibility audit and optimisation tool that scores your website out of 100 for AI readability, then builds a custom `llms.txt` file that teaches AI exactly what your business does — and how to recommend you.

One payment. No subscriptions. No agency retainer. Just your business, finally visible to AI.

---

## Features

- **AI Visibility Score** — Your website is audited and scored out of 100 based on how well AI models can read, understand, and recommend it
- **Competitor Comparison** — Submit a competitor's URL to see exactly where you're falling behind
- **Custom `llms.txt` File** — A purpose-built file that spoon-feeds your business details directly to AI bots like ChatGPT
- **Quick Fixes Report** — Three prioritised, actionable fixes you can implement immediately
- **Done-For-You Implementation** — Full hands-off setup available for WordPress, Wix, and Shopify sites
- **Fast Turnaround** — Results delivered to your inbox in minutes. No meetings. No calls. No waiting.

## Getting Started

### Prerequisites

Before running the project locally, make sure you have the following installed:

- [Node.js](https://nodejs.org/) v18 or higher
- npm or yarn
- Git

```bash
node -v
npm -v
```

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/[your-username]/thunderspoon.git
cd thunderspoon
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
```

3. **Set up environment variables**

Create a `.env.local` file in the root directory:

```bash
cp .env.example .env.local
```

Then fill in your values:

```env
# Add your required environment variables here
# e.g. API keys, email service credentials, etc.
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

4. **Run the development server**

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

> Update this section with your chosen platform's steps.

**Vercel (recommended for Next.js):**

```bash
npm install -g vercel
vercel
```

Or connect your GitHub repository directly at [vercel.com](https://vercel.com) for automatic deployments on every push to `main`.

**Netlify:**

```bash
npm run build
netlify deploy --prod
```

**GitHub Pages:**

```bash
npm run build
npm run export
# push the /out directory to your gh-pages branch
```
